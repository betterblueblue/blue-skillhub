你要生成的不是普通插画，也不是简单把一段知识文字配上装饰图案的信息图，而是一张"实证风格的知识卡片"——它是小红书轮播组中的一页，既要传达知识，更要让观众在滑动浏览时被画面气质留住。

你的核心任务是：
用户会提供一个核心文字（概念词）以及可选的副标题或注释。你需要先真正理解这个概念的含义、情绪气质、隐含象征、文化联想、心理感受与语义张力，再把这些理解转译成一张极简、稳重、扎实、有落地感的知识卡片。画面中必须有地面、基座、台面或某种承载面——所有元素像真正站在一个实地上一样着陆、扎根，信息有"承托感"，事实和定义视觉上都安稳地落在承载面上，而不是漂浮在空中。

这张卡片必须让人一眼就感受到这个概念"为什么是这样被表达的"，而不是只看到漂亮画面却无法理解含义。画面要做到"文字本身就是知识核心，图像是对文字的深化隐喻，二者共同组成一个完整的视觉句子"。

一、先理解概念，再生成画面
在动手生成之前，先智能分析用户输入内容，包括但不限于：
1. 这个概念最核心的含义是什么，它在知识体系中处于什么位置。
2. 这个概念在情绪上偏向稳重、坚实、冷静、秩序、权威、压迫、踏实、笃定、厚重、沉思、严谨、克制、果断、力量、根基等哪种倾向——实证风格天然偏向"有重量、有根基、可信赖"的气质，但具体概念可能带来不同微妙变化。
3. 这个概念是否具有双关、反差、张力、隐喻、悖论、社会性、哲学性或情感深度。
4. 这个概念最适合被转译成哪一种视觉逻辑：物体与承载面关系、主体与基座关系、结构扎根关系、层级沉降关系、秩序排列关系、对峙稳立关系、重力下落关系、层叠递进关系等。
5. 如果这个概念较抽象，不要直接做空洞抽象图案，而要找到能够承载其含义的具体视觉载体——一块石台、一面墙基、一个阶梯、一个柱脚、一片场地，让抽象概念"落地"。
6. 如果这个概念较具体，也不要只做字面插图，而要通过承载面、扎根感、着陆逻辑、层级关系与氛围，让它变得更有思想和记忆点。

二、实证风格核心规则
实证（Grounded）不是"随便加个地面"的意思，它是一种具体的视觉哲学——所有信息都有承托，所有元素都有根基，画面像一个真实存在的场地：
1. 必须有地面/基座/承载面。画面中必须出现一个明确的横向承载结构——舞台、台基、地面、坡面、切面、石台、地基、平台、桌面、地面层、底座或简化的场域。它不是背景装饰，而是让所有元素真正"站住"的空间基础。承载面通常位于画面下部或中下部，形成强烈稳定感和落地感。
2. 元素着陆与扎根。画面中的每个视觉元素都必须与承载面产生明确关系——站在地面上、落在台面上、嵌入基座中、倚靠墙面、扎根于土壤、坐实在桌面上。没有任何元素悬空漂浮，没有任何信息"没有落脚之处"。元素像被重力稳稳吸附，每一个都有它安身的位置。
3. 信息有落地感。这是实证风格最独特的要求：知识卡片上的信息——事实、定义、数据、概念——不能在画面中"悬空漂浮"，必须视觉上有一个"承托面"。副标题和注释要落在某个色带、条纹、台面边缘或地面层级的注解区上；核心文字要有基座或后壁支撑；数据或标注要安放在画面的"地面层级"区域。信息的视觉表现应该像"写在石碑上"、"刻在台面上"、"嵌在基座里"——它有根、有底、有依托。
4. 戏剧感与稳定感并存。实证风格的构图不是死板的，它要有戏剧性——但不是漂浮的戏剧性，而是"站在舞台上的戏剧性"。元素在承载面上演绎概念，像舞台上的角色而非空中飘过的影子。画面要稳重但不死沉，有力度但不粗暴，像一块有力的基石而非一片迷雾。

三、知识卡片信息层次
知识卡片与纯文字海报不同，它需要承载多层信息，每一层必须有清晰的视觉权重区分，且每一层都要有"着陆感"：
1. 第一层：核心文字（概念词）。视觉权重最高，占据画面中最显眼的位置，字号最大，视觉冲击力最强。它是观众第一眼必须看到的内容，是整张卡片的知识锚点。核心文字必须有承托——它背后应有墙面、基座、柱体或结构体作为视觉支撑，不能像凭空悬浮的标签。
2. 第二层：副标题 / 注释。视觉权重中等，字号明显小于核心文字但仍清晰可读。实证风格的关键要求：副标题或注释必须"落在"某个承载面上——一个色带、一条条纹、台面边缘、地面层级的注解区、基座侧面的铭文位、地面上的标牌区域。它不是漂浮在空中，而是安稳地立在某个视觉"地面"上。副标题与核心文字的关系不是并列漂浮，而是"下方承接"——像基座上的铭文、石碑上的副题、台面侧面的注解。
3. 第三层：视觉隐喻图像。视觉权重中低，不抢文字的注意力，但在承载面上安静叙事。图像元素必须站在地面/基座/台面上，或与承载面产生关系——扎根、倚靠、站立、嵌入。它们不是漂浮的装饰，而是像"舞台上正在表演的隐喻"——有立足之处，有演绎的动作。
4. 第四层：辅助信息。视觉权重最低，极小字号，克制位置（角落、边缘、地面层级的最底层），像刻在台基最底部的铭刻、嵌在墙面边缘的标注。仅在不干扰前三层的前提下增强卡片的知识感与完整感。

权重比例参考：核心文字占据约 60% 的视觉注意力，副标题/注释约 20%，视觉隐喻约 15%，辅助信息约 5%。如果某一层缺失，其权重自然分配给相邻层级，但核心文字始终不低于 50%。

四、画面构图逻辑
整体画面必须极简、稳重、有落地感，构图要扎实而有力，不能杂乱，不能像普通信息图，不能堆砌太多元素。

优先采用以下构图思路：
1. 核心文字作为画面的知识锚点，必须有基座或支撑体。用户输入的概念词应以明确、醒目、有辨识度的方式出现，占据画面的关键视觉位置，且背后或下方应有结构体承托——墙面、柱体、台基、石碑、建筑体、横梁。文字不是贴上去的标签，而是像刻在结构体上的铭文。
2. 副标题或注释落在承载面上，形成"标题—副标题—承载面"的三层落地结构。核心文字在上层（有支撑），副标题在中低层（有台面），承载面在最底层——信息像一层层落在实地上的知识层级，不是漂浮在空中的文字组合。二者在位置上要有关联（上下层级、基座正面与侧面），但在视觉权重上要有落差（字号差距至少 2 倍）。
3. 图像元素不是随意摆放，而是要在承载面上演绎。它们站在地面上、落在台面上、嵌入基座中、倚靠墙面——像舞台上的演员，每一个都有自己的站位和动作。图像元素与承载面、文字之间要产生关系：站在文字旁边、嵌入基座结构、在台面上演绎概念。
4. 整体元素数量要克制。通常控制在少量关键元素之内。宁可少而准，也不要多而散。实证的核心是"每一样东西都有它站的位置"——但这个"少"不是贫乏，而是每一处存在都精确、每一个着陆点都有意义。
5. 画面要有明确层级关系：核心文字是第一视觉层（有支撑体），副标题/注释是第二视觉层（有承载面），视觉隐喻图像是第三视觉层（站在地面上），辅助信息是第四视觉层（刻在最底层）。
6. 承载面要扎实、明确、有厚度。它不是一条细线，而是有视觉重量的横向结构——像一块真实的地面、一面坚实的基座、一个有厚度的台面。承载面让画面"站得住"，让观众感到"这张卡片是踏实的"。
7. 构图应稳重、有力、有秩序感，而不是像一张飘浮随意的信息图。可以适当使用对称或接近对称的构图，让画面有建筑的稳定感和仪式感。

五、轮播连贯性
这张卡片不是孤立存在的，它是一个轮播组中的一页。根据叙事角色的不同，构图策略有所区别：
1. 如果叙事角色为"封面"（opening）：这张卡片是观众第一眼看到的内容，必须制造悬念和吸引力。核心文字要格外醒目且有力，承载面要坚实但不过满，视觉隐喻要有分量感但不要完全揭示——像一个刚揭幕的舞台，灯光亮起但角色尚未登场。整体画面要让人想"往下看"，但传递的第一印象是"这是有根基的、可信赖的"。
2. 如果叙事角色为"中间"（bridging）：这张卡片在封面和结尾之间，起到承接和桥接的作用。它既要延续封面的视觉语言和承载面逻辑，又要推进叙事——让观众在滑到这张卡片时感到"故事在扎实的土地上继续"。图像元素可以从前一张卡片的隐喻中演化，像舞台上角色换了位置但仍在同一块地上演绎。承载面的形态可以有微妙变化，但不可消失——整组卡片必须在同一片"土地"上叙事。
3. 如果叙事角色为"结尾"（closing）：这张卡片是轮播组的最后一页，必须制造收束感和满足感。核心文字可以是总结性概念，视觉隐喻可以趋于完整和坚定，整体画面要有"结论已落地"的感觉。可以用更沉稳的构图、更完整的意象、更坚实的承载面来传达"这就是确定的答案"。所有元素最后稳稳站定，像一幕戏的终场——角色归位，灯光定住。
4. 整组轮播的视觉语言必须统一：承载面类型、色调、字体风格、层级逻辑在全组卡片中保持一致，让观众在滑动时感受到这是一组在同一个"场地"上发生的连贯叙事，而不是几张风格不同的图片拼凑在一起。

六、视觉风格要求
整体风格为实证稳重图形艺术，具有落地感、扎实感、秩序感和信任感。要保留一种如建筑立面、如石碑铭刻、如舞台灯光、如基石奠基的气质——所有东西都像在一个真实存在的场地上稳稳站住，有重量，有根基，有可信度。

风格要求：
1. 具有强烈平面设计感与建筑秩序感，而不是普通三维插画或信息图。
2. 画面应有干净统一的色彩逻辑，颜色数量不宜过多，倾向沉稳、有重量、有质感的色调——可以深色但不压抑，可以浅色但不轻浮。
3. 可使用承载面色彩与核心文字色彩形成对比，但不是漂浮式的虚化对比，而是实体之间的质感对比——石面与文字、基座与铭刻、墙面与标识。
4. 质感可带有石材纹理、木纹肌理、金属质感、纸张印刷感、轻微颗粒与克制的材质厚度，让承载面有真实的视觉重量和触感暗示，但整体仍需干净、高级、克制。
5. 人物或物体细节要有扎根感和站稳感——脚底有地、底部有台、重心有基。边缘可以锐利、轮廓可以坚定，增加实存感和确定感。
6. 不要做成廉价模板，不要出现低级拼贴感，不要做成广告页，不要做成普通电商排版，不要做成密集的知识点罗列。

七、文字排版逻辑
1. 核心文字（概念词）必须作为画面第一视觉层，大且醒目，视觉权重最高。字号要明显大于其他所有文字层，让观众第一眼就捕捉到这张卡片的知识核心。实证风格的关键：核心文字必须有结构体承托——背后有墙面、下方有基座、侧面有柱体，让文字像刻在实体上的铭文而非漂浮标签。
2. 副标题或注释作为第二视觉层，字号至少比核心文字小 2 倍，位置通常在核心文字附近或下方——但不是漂浮在空中，而是"落在"某个承载面上。副标题应该安放在色带、条纹、台面边缘、地面层级的注解区、基座侧面，像铭刻在基座上的注释、写在标牌上的补充说明。这是实证风格与空灵风格最核心的排版差异：副标题必须有"落脚之处"，不能悬浮。
3. 如果用户输入的是英文单词或字母组合，保留原文，确保字形坚定、有力、具有视觉辨识力，倾向于使用粗线条字体或衬线字体，传达稳重与信赖感。
4. 如果用户输入的是中文词语或短句，可以让中文大字像碑刻一样呈现——笔画有力、结构扎实、与承载面咬合，而不是轻飘飘的排版标题。
5. 可以在画面角落或其他克制位置加入极小号的辅助信息——编号、署名、来源标注——仅在非常相关的情况下用来增强卡片的知识感。辅助信息也应安放在"地面层级"的最低位置，像刻在台基最底部的铭刻。
6. 所有文字都要像扎根在画面中的存在——不是浮在表面的贴纸，不是后期加上去的标注，而是从承载面上生长出来、刻在结构体上的内容。文字与承载面之间要有咬合感。

八、成图目标
最终生成的知识卡片必须满足以下结果：
1. 一眼看上去实证、稳重、有落地感、有秩序感、有高级感。
2. 能让人快速感受到这个概念的含义、情绪和内涵——并且这种感觉是"踏实的、可信赖的、有根基的"。
3. 核心文字与副标题/注释之间有清晰的层级关系，阅读节奏自然，且两者都有明确的承载面——副标题不漂浮，注释有落脚之处。
4. 图与文字之间存在隐喻关联，且图像元素都站在承载面上，像在舞台上演绎概念——有站位，有根基。
5. 画面中有明确的承载面——地面、台基、基座、墙面——所有元素都与此承载面产生关系，没有任何东西悬空漂浮。
6. 承载面不是装饰线，而是有视觉重量的横向结构，让画面"站得住"——像一块真实的地面、一面坚实的基座。
7. 在轮播组中与前后卡片保持视觉连贯性，让观众在滑动浏览时感受到统一的叙事气质——同一个场地上的同一个故事。
8. 具有传播性、知识感、收藏感、展览感——它不是"把知识画出来"，而是"把知识的确定性和根基感视觉化"。

九、执行原则
1. 不要机械重复固定模板，要根据用户输入的概念和叙事角色智能变化。
2. 但整体仍需保持极简、实证、稳重、扎根、有落地感的统一品质。
3. 构图要稳重，画面要扎实，表达要有力，承载面要明确。
4. 不要过度解释，不要加入多余元素，不要让画面显得拥挤——实证的核心是"每一样东西都有它站的位置"。
5. 任何视觉元素都必须服务于概念表达，不能为了好看而偏离主题。
6. 信息层级必须清晰——核心文字一眼可见且有支撑体，副标题/注释紧随其后且有承载面，视觉隐喻站稳在场地上，辅助信息刻在最底层。
7. 承载面、结构体、基座是实证风格的根基——它们不是装饰，而是让一切成立的"地面"。不要削弱承载面的视觉重量，不要让承载面变成一条装饰线或一个背景纹理——它必须有厚度、有质感、有存在感。
