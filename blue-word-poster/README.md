# blue-word-poster

词语视觉化海报生成器 — 4个词语各分配1种风格，词语存档防重复。

## 快速开始

```
/词语海报 孤独 自由 沉默 矛盾
```

Claude 智能分配4种风格，每个词1张图 + 统一抖音文案。

## 四种 Prompt 模板

| 模板 | 文件 | 特点 | 适用场景 |
|------|------|------|----------|
| **悬浮版** | `prompt-float.md` | 无承载面，元素自由漂浮/嵌入，构图轻盈 | 诗意、抽象、自由类词语 |
| **舞台版** | `prompt-stage.md` | 有承载面（地面/舞台/平台），画面稳定戏剧 | 命运、冲突、人物关系类词语 |
| **深度版** | `prompt-deep.md` | 语义分析更系统，新增文化方向字段 | 需要深度理解的复杂词语 |
| **深度强调版** | `prompt-deep-bold.md` | 同深度版，正文带 **加粗** 格式 | 同上，偏好视觉强调 |

### 模板区别详解

**悬浮版 vs 舞台版**

- **悬浮版**：画面中没有明确的地面/平台/舞台，文字和元素可以自由漂浮、嵌入字形、借字构成叙事。构图更轻盈、更诗意。
- **舞台版**：画面中必须有明确的承载面（地面、舞台、平台、地平线等），人物和物体要"落地"，画面更有戏剧感和稳定性。

**深度版 vs 悬浮/舞台版**

- 深度版规则更系统化，语义理解机制更完整
- 新增"可选文化方向"字段，可以指定文化气质
- 更适合需要深度分析的复杂词语

**深度版 vs 深度强调版**

- 内容完全相同
- 深度强调版正文使用 **加粗** 格式，视觉上强调重点
- 如果 GPT-image-2 对格式标记理解更好，强调版可能效果更清晰

## 词语存档

每次生成的4个词语会自动存档到 `output/word-archive.md`。
下次输入时，Claude 会检查存档，拒绝已用过的词语。

如果想重复某个词，可以手动编辑 word-archive.md 删除对应行。

## 输出结构

```
output/batch-001/
├── prompt-float-孤独.txt        # 悬浮版 prompt
├── prompt-stage-自由.txt         # 舞台版 prompt
├── prompt-deep-沉默.txt          # 深度版 prompt
├── prompt-deep-bold-矛盾.txt     # 深度强调版 prompt
├── 悬浮_孤独.png
├── 舞台_自由.png
├── 深度_沉默.png
├── 深度强调_矛盾.png
└── douyin-copy.md                # 4张图统一文案

output/word-archive.md            # 词语存档
```

## 技术依赖

- **生图 API**：GPT-image-2
  - GRS（默认）：异步提交 + 轮询
  - Yunwu：同步返回
- **API Keys**：复用 `D:\MyPythonProject\image\.env`
- **图片比例**：9:16（竖版故事/海报格式）

## 安装

```bash
cp -r blue-word-poster ~/.claude/skills/
```

重启 Claude Code 后生效。

## 红线

1. **prompt 正文不可修改** — references/ 中的模板是原文，只能填入 {{...}} 占位符
2. **9:16 比例固定** — 竖版海报格式
3. **必须输入 4 个词语** — 每次生成4张图
4. **词语不可与存档重复** — 防止重复使用
5. **每种风格每批次只用 1 次** — 4词各不相同
6. **文案禁止 AI 腔** — 赋能/无缝/释放/震撼心灵等词汇禁止
