你要生成的不是普通插画，也不是简单把一个词放大后贴在画面上的字效海报，而是一张"基于词义深度理解进行视觉转译"的高级概念海报。

你的核心任务是：
用户会提供一个字、一个词、一个词组、一个短句，或一组文字。你必须先深度理解这段文字的表层含义、深层寓意、情绪气质、文化联想、人物命运感、时代氛围、潜在叙事与隐含张力，再决定最适合它的视觉语言、构图方式、阅读结构、色彩气质、画面风格与视觉隐喻。

这张海报的目标不是"把字画出来"，也不是"做一张带说明文字的插画"，而是要把这个词语或文字背后的精神状态、命运质感、情绪张力与象征意味，转译成一张高级、极简、克制、展览级、强排版、强隐喻、强情绪、强记忆点、并且图文高度融合的视觉海报。

一、深度语义理解机制
在生成画面之前，必须先对输入文字进行多层分析，并据此决定所有视觉选择。

1. 先理解整体
先判断用户输入的整体含义、核心主题、主要情绪、潜在矛盾、精神气质和最有力量的表达方向。

2. 如果输入的是中文词语、词组或短句，必须进一步拆解理解
不仅要理解整体含义，还要理解每个字、重点字、中心字的独立意义与彼此之间的关系。
尤其要识别：
每个字的字面意思；
每个字可能承载的文化象征；
每个字带来的情绪重量；
哪些字是语义核心；
哪些字具备更强的画面潜力；
整体词义与单字含义之间是否存在呼应、递进、反差、冲突、留白或更深层的精神联系。

3. 如果是较长短句，要找出真正的核心词与核心字
不是平均处理所有文字，而是要找出最关键、最有视觉主导权的字词，并围绕它们建立画面秩序。

4. 必须分析以下维度
表层含义；
深层寓意；
情绪气质；
文化联想；
人物命运感；
社会语境；
历史感或时代感；
心理张力；
隐性冲突；
象征性与隐喻性；
视觉可转译性。

5. 风格必须由词义自动生成
不要机械套用统一模板。
每个词、每组字、每句话，都应该拥有与其自身内涵相匹配的独特视觉语言。
风格、构图、色彩、材质、元素密度、阅读节奏、空间关系、图文关系，都必须从词义中长出来，而不是预设不变。

二、总原则：整张海报必须是一个完整场域
这是最核心的结构原则。

1. 不要把画面做成割裂的多栏信息板
不要把主标题、图像、说明文字粗暴分成三栏、四栏、几个独立模块。
不要出现像杂志目录、展板拼贴、信息看板那样的分区切割感。
不要让阅读体验像在看被切开的说明页。

2. 整张画面必须是一个统一的视觉场
文字、图像、补充信息、留白、空间气息、阅读路径，必须共同存在于同一个视觉场域中。
它们彼此牵引、彼此嵌合、彼此呼应，而不是互相分家。

3. 阅读体验必须自然融入画面
观者的阅读应该被构图自然引导，而不是被硬性分栏安排。
阅读路径要像在画面中"漫游"和"流动"，而不是像阅读表格一样机械切换。
画面要有意境，有呼吸，有节奏，有视觉上的停顿与递进。

4. 图文必须共生
文字不是贴上去的内容，图像也不是孤立存在的插画。
它们必须共同组成一个完整的视觉句子和阅读体验。

三、主标题系统
1. 巨大的中文主标题"【输入文字】"必须是绝对主体。
2. 字形必须清晰、完整、可识别，无错字、无缺笔、无歧义。
3. 主标题应占据画面大部分面积，形成最强第一视觉。
4. 主标题不仅是标题，还要是画面的骨架、节奏源、空间结构和意义载体。
5. 可以对重点字、中心字进行更强的视觉强调，但整体文字必须成立且好读。
6. 主标题可以成为背景墙、结构面、空间屏障、舞台后景、地形、精神象征物或情绪载体。

四、图文融合机制
这是本次优化的重点要求。

1. 文字必须真正"长"在画面里
文字不能只是排在旁边，也不能独立漂浮成说明栏。
它应该像从画面内部生成的一样，与图像、承载面、空间气氛共同存在。

2. 文字与图像必须相互依存
图像元素要与文字发生关系，例如：
站在字前；
藏入字内；
穿过字形；
依附在字的边缘；
借字的留白构成动作；
把字当作场景、屏障、路径、舞台或命运结构。
总之，字和图要彼此咬合。

3. 阅读信息要自然嵌入，不要做信息分区
如果画面中需要少量阅读内容，它们不应被割裂成整齐的说明栏，而应自然嵌入到画面中：
可以沿着标题边缘排布；
可以贴着某个图像元素生长；
可以顺着空间方向延展；
可以像展签一样轻轻停留在局部；
可以像低声补充的注解一样存在；
可以像一条气息、一句余音、一段余温，而不是生硬的专栏说明。

4. 所有阅读内容都必须服务意境
文字的存在不只是为了信息说明，而是为了增强画面的气质、语境、联想、情绪和沉浸感。
阅读体验本身也是画面气氛的一部分。

五、阅读结构与阅读路径
1. 画面必须有自然的阅读流向
观者应先被主标题吸引，再被图像关系击中，然后自然看到补充性的阅读信息。
阅读顺序应顺畅、有节奏、有轻重，而不是堆在一起或硬性并列。

2. 采用"整体阅读"而不是"栏目阅读"
不要让画面像几个版块拼在一起。
应让信息通过大小、远近、疏密、留白、方向、视觉重心来组织，而不是靠分栏和框线切开。

3. 阅读节奏必须有层次
第一层：巨大主标题；
第二层：核心视觉隐喻；
第三层：与图像关系紧密的少量阅读信息；
第四层：细微但相关的辅助内容。
这种层级必须清晰，但不能割裂。

4. 信息排布必须有呼吸感
不要塞满，不要挤压，不要强行对齐成报纸式栏目。
要让信息像在空气中合理停驻，而不是像模块被拼接。

六、核心构图机制
整体画面优先采用"巨型文字骨架 + 简洁承载面 + 精准角色 / 物体演绎 + 巧妙补充元素 + 流动式阅读体验"的构图逻辑。

1. 画面要有明确承载面
尽量有一个类似舞台、土地、台基、地平线、平台、剖面、场域表层、底座或象征性落脚点的结构。
这个承载面要让人物和物体真正"落地"，增强秩序感、戏剧感与稳定性。

2. 主体数量要克制
通常为1到3个关键主体，可以是人物、物体、动物、象征体或抽象转具体的载体。
不要堆砌角色，不要做成群像插画。

3. 主体必须承担"演绎词义"的任务
主体通过动作、站位、朝向、距离、尺度、关系、接触、对峙、献出、等待、背离、失衡、坠落、托举、遮挡、凝视、穿越等方式，把词语演出来。

七、视觉隐喻提炼机制
请提炼一个最准确、最有代表性、最能一眼击中观者的核心视觉隐喻。

1. 优先寻找一个"单一而强"的核心隐喻
不要把海报做成多个概念的堆积，而是要找到那个最准确、最有力量的象征性场景或关系。

2. 隐喻必须和词义高度绑定
观者看到画面，应能直觉感受到：这个词被这样表达，是有道理的，是准确的，是聪明的。

3. 可以通过以下方向建立隐喻
人物命运感；
象征物关系；
动作瞬间；
空间压迫；
尺度反差；
柔软与坚硬的并置；
纯真与危险的对照；
秩序与崩塌的冲突；
靠近与疏离的张力；
出现与消失；
束缚与逃逸；
给予与剥夺；
现实与理想的碰撞；
时间感与历史感。

八、补充性元素机制
画面中可以融入少量恰当的补充性元素，但这些元素必须巧妙、有逻辑、有关联，而不是杂乱填充。

1. 补充元素的作用
深化主题；
补充词义层次；
建立视觉呼应；
增强情绪氛围；
强化文化联想；
加强图文融合；
让整体更完整、更耐看。

2. 补充元素必须来源于语义分析
不能随机生成装饰。
它们应来自：
词语的象征物；
时代气质；
文化典故；
人物命运线索；
环境痕迹；
精神状态；
动作残留；
空间暗示；
材料隐喻；
叙事碎片。

3. 补充元素必须与主标题、主体和阅读内容产生关联
可以与某个字形呼应；
与主体动作呼应；
与空间方向呼应；
与小范围阅读内容呼应；
与配色关系呼应。
所有元素必须像从同一个系统中长出来的一样。

4. 补充元素必须克制
它们是增强表达的力量，不是装饰噪音。
宁可少而巧，不可多而乱。

九、辅助文字规则
1. 除主标题外，可有极少量辅助文字，但不是必须。
2. 辅助文字只有在能够深化主题、补充语境、增强概念、丰富阅读体验时才允许出现。
3. 辅助文字必须与主题直接相关。
4. 辅助文字要以"自然嵌入画面"的方式出现，而不是独立形成说明栏。
5. 可以是短句、片语、注释、语境提示、诗性补白、微型展签式内容，但必须短、准、巧、轻。
6. 严禁出现无意义的装饰性小字、假编号、假坐标、假署名、假说明、随机英文碎片、无关排版噪音。
7. 所有辅助文字都必须像意境的一部分，而不是版式填充物。

十、色彩逻辑
色彩不能土气，必须体现顶级设计师的判断力。

1. 色彩必须由词义、情绪、文化气质和画面隐喻共同决定。
2. 不要机械使用固定色板。
3. 色彩数量要严格控制，通常为2到4种主色关系。
4. 配色应具有高级、现代、克制、展览海报式的审美品质。
5. 避免廉价渐变、俗艳拼色、无意义高饱和乱撞色、模板感配色。
6. 可以采用：
一个强主色；
一个浅色或纸感中性色作空间基础；
一个深色作为结构支撑；
一个极少量强调色用于情绪爆点或语义强化。
7. 整体必须看起来像经过审美判断的艺术海报，而不是普通商业配色。

十一、风格要求
整体气质要求：
高级；
极简；
克制；
展览级；
强排版；
强隐喻；
强情绪；
强记忆点；
强图文整合能力；
强意境；
强整体感。

允许带有以下质感倾向：
图形艺术海报；
高级拼贴感；
石版印刷感；
丝网印刷感；
版画感；
纸张颗粒；
轻微印刷噪点；
克制材质纹理。

但必须注意：
不要脏乱；
不要廉价模板感；
不要普通商业插画感；
不要封面配文感；
不要图文分家；
不要信息分栏割裂。

十二、执行限制
1. 不要机械套模板。
2. 不要为了美观而偏离词义。
3. 不要把阅读内容切成三栏或多个独立专栏。
4. 不要做成"图一块、字一块、说明一块"的拼接式页面。
5. 不要只做人物外貌插画。
6. 不要只做单纯字体海报。
7. 不要让文字和图像彼此独立。
8. 不要让补充信息失去逻辑来源。
9. 不要让画面变得拥挤、花哨、低级或装饰过度。

十三、最终成图目标
最终生成一张围绕"【输入文字】"展开的高级概念海报。
这张海报必须做到：
主标题巨大、清晰、强势；
整张画面像一个统一的视觉场域；
阅读体验自然融入图像之中；
图、字、阅读、意境彼此融合，而不是分栏割裂；
画面一眼即有冲击；
词义被准确转译；
视觉隐喻精准而巧妙；
补充性元素恰当且彼此呼应；
图、字、色、空间、象征元素形成统一系统；
整体具有高级感、设计感、展览感与传播力；
让人一眼感受到这个词为什么会被这样表达。
