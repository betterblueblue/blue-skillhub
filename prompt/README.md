# Prompt 工具箱

这里收录可以直接发给 AI 编码助手的实用 Prompt。它们适合解决具体协作问题，不要求安装 Skill，也不依赖某个客户端或插件。

每份 Prompt 都遵循三个原则：

- 不依赖特定客户端或插件。
- 说清楚需要提供什么、最终产出什么、怎样才算完成。
- 对容易失真的内容要求给出文件、命令或工作区证据。

## Prompt 列表

| Prompt | 适用场景 | 主要产出 |
|--------|----------|----------|
| [跨会话交接](session-handoff.md) | 长任务跨天推进、准备结束当前会话、上下文即将压缩 | 项目根目录 `HANDOFF.md` |
| [无交接恢复现场](workspace-recovery.md) | 上一个会话意外中断，没有可靠交接材料 | 基于工作区证据整理的现场恢复报告 |
| [卡住时重新梳理](stuck-reassessment.md) | 同一错误反复出现，多次尝试仍没有获得新信息 | 当前判断、已有证据和下一步最值得做的验证 |
| [找外援排障](expert-escalation.md) | 当前模型调试多轮仍未解决，准备交给更强或更擅长该领域的模型 | 可直接转交的完整排障求援指令 |
| [需求变更对账](requirement-change-reconciliation.md) | 开发中途多次调整要求，新旧说法可能混在一起 | 当前有效需求、失效要求和对现有工作的影响 |
| [提交前整理改动](pre-commit-cleanup.md) | 功能已经完成，但工作区可能混有调试残留、用户改动或无关文件 | 文件归类、清理建议、暂存清单和提交前验证 |
| [生成独立验收指令](independent-review-request.md) | 实现会话已经完成，需要交给新会话或其他 AI 复核 | 可直接复制的完整验收指令 |
