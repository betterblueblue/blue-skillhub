# Prompt 工具箱

这里存放可以直接复制给 AI coding agent 的实用 Prompt。

每个 Prompt 尽量满足三点：

- 不依赖特定客户端或插件。
- 明确输入、输出和完成标准。
- 对容易失真的内容要求给出文件、命令或工作区证据。

## Prompt 列表

| Prompt | 适用场景 | 主要产出 |
|--------|----------|----------|
| [跨会话交接](session-handoff.md) | 长任务跨天推进、准备结束当前会话、上下文即将压缩 | 项目根目录 `HANDOFF.md` |
| [卡住时重新梳理](stuck-reassessment.md) | 同一错误连续出现、连续尝试没有获得新信息 | 有证据的当前判断和下一项最小验证 |
| [生成独立验收指令](independent-review-request.md) | 实现会话已经完成，需要交给新会话或其他 agent 复核 | 可直接复制的完整验收指令 |
